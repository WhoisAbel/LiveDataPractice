# mvvmNewsApp

**News App Tutorial on YouTube.**

https://youtu.be/asuOWE5KuFM

In this project, you will learn how to build an Android app using the Model-View-ViewModel presentation architecture pattern.

Technologies:

- Kotlin
  
- MVVM Design Patteren
  
- Room
  
- ViewModel
  
- LiveData
  
- Coroutine
  
- Navigation
  
- DataBinding
  
- Retrofit
  
- Gson
  
- DiffUtil
